# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-15

### Added
- Initial production release
- Comprehensive Playwright integration test suite
- CI/CD pipeline with GitHub Actions
- Build system for generating Tampermonkey userscript
- Complete documentation
- Project structure with src/, dist/, tests/ directories
- Test helpers for script injection and keyboard simulation
- Support for multiple browsers (Chromium, Firefox, WebKit)

### Features
- Escalation logic with hotkey '3'
- Manual reset with hotkey '0'
- Closed-loop pressure controller
- Real-time HUD display
- Audio and visual feedback
- Custom event system
- Configurable parameters

### Testing
- 40+ integration tests covering all functionality
- Script injection and initialization tests
- Escalation logic tests
- Pressure controller tests
- Audio/visual feedback tests
- HUD functionality tests
- Reset functionality tests
- Event system tests
- Edge case handling tests

### Infrastructure
- Playwright test framework
- GitHub Actions CI/CD
- Automated browser testing
- Test result artifacts
- Build script for distribution
- ESLint and Prettier configuration
